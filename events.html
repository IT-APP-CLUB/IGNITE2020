Clash of Coders

It is a programming event to test the fundamentals of the most popular programming languages of all participants. The rounds will start with the basic concepts and slowly buildup to challenge their programming and logic application skills. This event will help developers to test and re-evaluate their basics and beginners to solidify their concepts. The languages permitted in this event are C, C++, java and Python.

Registration fees: ₹ 200.

Prize Worth: ₹ 5,000/-.

● Participant cannot use any other language than the two he has already selected.
● Participant has to select 2 languages for quiz round. Other rounds will be either in C or C++.
● Do not generate the entire pattern using only ‘printf’ or ‘cout’ statements. Use your logic.
● Internet access is prohibited for referring code solution.
● Generated pattern should be exactly the same as given in the problem statement.
● If anyone is found violating these rules, they will be immediately disqualified from the event.

Round 1: Quiz
● All participants will have to choose 2 languages from C, C++, Java and Python.
● Each participant will be given 100 randomly selected MCQ questions based on their selected programming languages to solve within 30 minutes.
● On the basis of how many questions are answered correctly, the participant’s performance will be evaluated.
● After grading all students, the top 50% will be selected for the next round.

Round 2: Pattern Generation
● Each participant will be given a pattern which they will have to generate using the programming language specified in the problem statement and will have to complete this task within 30 minutes.
● The participants will be evaluated on the basis of their completion speed and the top 50% will be selected for the final round.

Round 3: Advanced programming
● Each participant has to pick a chit which contains a problem statement and they will have to satisfy all necessary objectives given using programming language specified in the chit.
● The first participant who successfully implements all modules of the problem statement correctly will be declared the winner.
